Funcionalidades do Site

Este site oferece várias ferramentas para explorar os dados da Copa do Mundo de 2022, incluindo:

  • Visualização completa da tabela de grupos com classificação das equipes.
  • Pesquisa de partidas por país e datas específicas, com todos os detalhes das partidas.
  • Exibição de estatísticas detalhadas das partidas
  • Filtragem interativa para encontrar informações rapidamente.

Todos os dados são fornecidos pela API da Copa do Mundo, garantindo informações precisas e atualizadas.

Sobre o Projeto

Este site foi desenvolvido como parte de um trabalho prático da disciplina de Programação Estruturada e Orientada a Objetos (PEOO). O principal objetivo é aplicar conceitos de manipulação de objetos, JSON e integração com APIs para construir aplicações web práticas e interativas.

O projeto utiliza dados reais da API da Copa do Mundo para fornecer uma experiência interativa ao usuário, permitindo a navegação e pesquisa de informações de maneira dinâmica. Link para a api World Cup API
